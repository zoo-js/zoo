## 💄 如何领养？

新开一个 [issue][issue-new]，填写自己想要领养（加入）小萌宠。同时原则上仅支持单人领养 **5只** 小宠物，请大家谨慎挑选。一经挑选，请勿随意抛弃。谢谢大家！
